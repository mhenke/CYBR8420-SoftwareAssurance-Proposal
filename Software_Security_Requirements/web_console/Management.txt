@startuml
actor Administrator

rectangle "Keycloak Admin Console" {
    usecase "Manage Users, Roles, Clients, and Realms" as ManageMulti
    usecase "Configure Security Settings" as ConfigureSecurity
}

Administrator --> ManageMulti: Create/Update/Delete Users, Roles, Clients, and Realms
Administrator --> ConfigureSecurity: Adjust security-related configurations

ManageMulti --> Administrator: Realms updated
ConfigureSecurity --> Administrator: Security settings applied
